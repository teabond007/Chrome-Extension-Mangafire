Bookmark Color Marker is a Chrome Extension that allows you to add custom bookmarks to MangaFire.to

If you have a custom Bookmark add it here because the default statuses are (reading, completed, dropped, on-hold, plan to read, read(which is depriorotized))

When Auto Sync is enabled it will sync Bookmarks every set days.

After the day mark is hit it will wait until Mangafire is opened, then Sync and then reset the counter

When Custom Border Size is enabled it will change the thicness of the marker

Suggested size is anywhere between 2 - 8

Sync Button Requieres you to be logged in Mnagafire.to

Sync and Mark Read also saves/highlights your history/read mangas.

if Sync and Mark Read is disabled syncing bookmarks will not save read mangas, after enabling this option you have to sync bookmarks again

Works only on Mnagafire.to
